# 美丽省钱罐 - 项目开发计划

## 项目概述
- **项目名称**：美丽省钱罐
- **定位**：女性向省钱游戏小程序
- **核心价值**：通过记录省钱行为实现变美梦想
- **开发周期**：3天MVP，1周完整版

## 技术架构
### 前端
- 微信小程序原生开发
- 使用云开发能力
- UI框架：ColorUI或Vant Weapp

### 后端
- 微信云开发（云函数、云数据库、云存储）
- 无需独立服务器

### 数据库设计
1. **用户表**：openid、昵称、头像、注册时间
2. **梦想表**：用户id、梦想名称、目标金额、当前进度、创建时间
3. **省钱记录表**：用户id、省钱金额、省钱描述、记录时间、关联梦想
4. **积分表**：用户id、总积分、等级、成就徽章

## 页面结构
1. **首页**：梦想进度总览、快速记录入口
2. **梦想管理页**：创建/编辑/删除梦想
3. **记录历史页**：省钱记录列表
4. **社交页**：闺蜜列表、排行榜
5. **我的页**：个人中心、成就徽章、设置

## 核心功能点
### 1. 梦想创建
- 选择预设梦想（医美、包包、旅行等）
- 自定义梦想名称和图片
- 设置目标金额

### 2. 省钱记录
- 快速记录：预设场景（没喝奶茶、自带午饭等）
- 详细记录：自定义金额和描述
- 关联梦想：记录时选择关联的梦想

### 3. 进度可视化
- 进度条动画
- 梦想图片渐变效果
- 距离目标倒计时

### 4. 游戏化系统
- 积分计算规则
- 等级成长体系
- 成就徽章解锁
- 每日任务

### 5. 社交功能
- 闺蜜邀请与认证
- 组队省钱挑战
- 排行榜（个人、闺蜜、城市）
- 省钱经验分享
